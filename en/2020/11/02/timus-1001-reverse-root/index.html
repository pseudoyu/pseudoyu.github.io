Timus 1001 Reverse Root

Problem Description

The problem is so easy, that the authors were lazy to write a statement for it!

Input: The input stream contains a set of integer numbers Ai. The numbers are separated by any number of spaces and line breaks. A size of the input stream does not exceed 256 KB.

Output: For each number Ai from the last one till the first one you should output its square root. Each square root should be printed in a separate line with at least four digits after decimal point.

My Solution

import java.util.*;

public class problem1001 {
   public static void main(String[] args) {
       ArrayList<Long> list = new ArrayList<>();
       Scanner in = new Scanner(System.in);
       while (in.hasNextLong()) {
           long e = in.nextLong();
           list.add(e);
       }
       in.close();

       for (int i = list.size() - 1; i >= 0; i--) {
           System.out.printf("%.4f%n", Math.sqrt((double) list.get(i)));
       }
   }
}
