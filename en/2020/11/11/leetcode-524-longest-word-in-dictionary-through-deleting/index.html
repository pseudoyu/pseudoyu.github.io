LeetCode #524 Longest Word in Dictionary through Deleting

Problem Description

Given a string and a string dictionary, find the longest string in the dictionary that can be formed by deleting some characters of the given string. If there are more than one possible results, return the longest word with the smallest lexicographical order. If there is no possible result, return the empty string.

Example 1:

Input:
s = "abpcplea", d = ["ale","apple","monkey","plea"]

Output: 
"apple"

Example 2:

Input:
s = "abpcplea", d = ["a","b","c"]

Output: 
"a"

My Solution

这道题主要也是考察通过两个指针来判断一个字符串是否是另一个字符串的子序列，可以分为两个函数来进行判断，逻辑实现会更加清楚。首先构造判断子字符串的函数，两个指针向右遍历，遍历大字符串的指针向右逐次移动，当两个指针指向值相等时，小字符串的指针向右移动一位，最后判断移动位数是否和小字符串的相等。然后我们开始撰写找最长子字符串的函数主体部分，遍历给定子字符串列表中的所有元素，比较当前字符和结果字符，调用函数判断是否为子字符串，遍历完成后返回结果。

class Solution {
    public String findLongestWord(String s, List<String> d) {
		// 初始化一个字符串保存结果
		String result = "";
		
		// 遍历d列表中当个元素
		for (String t:d) {
			// 比较当前字符与结果字符
			if (result.length() < t.length() || (result.length() == t.length() && result.compareTo(t) > 0)) {
				if (isSubstr(t, s)) {
					result = t;
				}
			}
		}
		return result;
	}

	// 构造一个判断是否为子字符串的函数
	private boolean isSubstr(String t, String s) {
		int i = 0;
		int j = 0;

		// 两个指针逐个遍历，i向右，当相等时j向右
		while (i < s.length() && j < t.length()) {
			if (s.charAt(i) == t.charAt(j)) {
				j++;
			}
			i++;
		}

		// 判断字符串长度是否相等
		return j == t.length();
	}
}
