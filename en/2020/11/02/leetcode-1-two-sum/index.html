LeetCode #1 Two Sum

Problem Description

Given an array of integers nums and an integer target, return indices of the two numbers such that they add up to target.

You may assume that each input would have exactly one solution, and you may not use the same element twice.

You can return the answer in any order.

Example 1:

Input: nums = [2,7,11,15], target = 9
Output: [0,1]
Output: Because nums[0] + nums[1] == 9, we return [0, 1].

Example 2:

Input: nums = [3,2,4], target = 6
Output: [1,2]

Example 3:

Input: nums = [3,3], target = 6
Output: [0,1]

My Solution

Brute Force Solution

第一种就是暴力解法，两层循环嵌套遍历所有结果，满足条件即返回，思路简单但运行会比较慢

class Solution {
    public int[] twoSum(int[] nums, int target) {
        //初始化存储结果的数组
        int[] result = new int[2];

        //前->后嵌套遍历nums数组
        for (int i = 0; i < nums.length; i++) {
            for (int j = i + 1; j < nums.length; j++) {
                
                //试探所有可能性，如满足条件则返回两个索引，得到结果
                if (nums[i] + nums[j] == target) {
                    result[0] = i;
                    result[1] = j;
                    return result;
                }
            }
        }
        return result;
    }
}

HashMap Solution

第二种就是利用HashMap来进行解答，这种解法是用了HashMap数据结构，前->后遍历整个数组，将nums数组转为<值，索引>的HashMap，第一个值由遍历生成，在HashMap里使用containsKey方法找另一个值，找到后返回索引，把两个索引返回即可。因为HashMap可以直接键值配对，会大大提高效率

class Solution {
    public int[] twoSum(int[] nums, int target) {
        //初始化存储结果的数组
        int[] result = new int[2];

        //初始化一个存放(值, 索引）的HashMap
        HashMap<Integer, Integer> numsMap = new HashMap<Integer, Integer>();
        
        //前->后遍历nums数组
        for (int i = 0; i < nums.length; i++) {

            //第一个数由循环遍历生成，求出要达成条件的另外一个数的值
            int remainNum = target - nums[i];
            
            //当找到了另一个数时，求出这个键的值（即索引）
            if (numsMap.containsKey(remainNum)) {
                result[0] = numsMap.get(remainNum);
                result[1] = i;
                return result; 
            }
            
            //将nums数组的值和索引一一对应放到numsMap里
            numsMap.put(nums[i], i);
        }
        return result;
    }
}
