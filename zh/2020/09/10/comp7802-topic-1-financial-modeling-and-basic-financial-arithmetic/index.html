Topic 1.2 Basic Financial Arithmetic

Interest Rate

The period for which the investment/loan will last (e.g. 10% for 6-month)

Interest is calculated on the following year basis

  • Hong Kong Dollar - 365 days or 366 days (in leap year)
  • Pound Sterling, Singapore Dollar, Thai Baht - 365 days
  • Others - 360 days

Simple Interest

Total_Proceed = Principal * (1 + interest_rate * (days / year))
$104 = $100 * (1 +4% * (365 / 365))

Compound Interest

Total_Proceed = Principal * (1 + interest_rate * (days / year))^N
$108.16 = $100 * (1 +4% * (365 / 365))^2

Nominal and Effective Rates

Total_Return = Principal * (1 + (nominal_rate / n))^n
1 + effective_rate = (1 + nominal_rate / n) ^ n
effective_rate = [(1 + nominal_rate / n) ^ n - 1]
nominal_rate = [(1 + effective_rate)^(1 / n) - 1] * n

Annually Compound Rates

annually_compound_rate = [(1 + nominal_rate * (days / year))^(365 / days) - 1]

Interest = (1 + annually_compound_rate)^(days / 365) - 1
Interest = (1 + nominal_rate * (days / year)) - 1

Compound Yield

compound_yield = (1 + nominal_rate * (days / year))^(365 / days) - 1

Daily Compound Rates

daily_compound_rate = [(1 + annually_compound_rate)^(1/365) - 1] * 365

Continuous Compound Rates

r is the continuously compounded rate
i is the nominal rate for a year

r = ln(1 + i)
i = e^r - 1
